# Super Mario Bros. 3

## What is this?

This is a work-in-progress Archipelago integration for **Super Mario Bros. 3**.

The current implementation started as a very small vertical slice focused on **World 1** and now includes a first pass at SRAM-based clear checks for later overworld maps.

## Current Scope

- World 1 access/goal logic
- SRAM-based clear checks for numbered levels, fortresses, and the World 2 pyramid
- Goal is still clearing World 1 Castle
- Minimal item pool
- BizHawk-based play is planned
