# Super Mario Bros. 3

## What is this?

This is a work-in-progress Archipelago integration for **Super Mario Bros. 3**.

The current implementation is a very small vertical slice focused only on **World 1 & 2**.

## Current Scope

- World 2 only
- Goal is clearing World 2 Castle
- Minimal item pool

## Known issues
- Logic is messed up for castle/fortress access unlocks
- Because it's untested, check memory addresses might be screwed up as well for world 2

## Planned Features
- Rom patch to allow for more options (warping, saving, level rando, etc.)
- Remaining worlds
- YAML options
